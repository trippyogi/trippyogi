## Tools

Agent-operations tooling, extracted from running a real company on agents.

- **[Baton](https://github.com/trippyogi/baton)** - a ranking engine for human attention in multi-agent systems. Watches tasks, runs, and agents, then ranks the single human touch that unblocks the most agent work, each with a plain-English reason. Runs my company's daily operations.
- **[gitworthy](https://github.com/trippyogi/gitworthy)** - is it worth your commit? Pre-flight triage for OSS contribution targets: checks whether an issue is already done, in flight, fixed but unreleased, duplicated, or genuinely open. CLI + MCP server, on [npm](https://www.npmjs.com/package/gitworthy).
